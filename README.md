# Meridian

Coming-soon landing for a specialty coffee brand. Astro 5 + Tailwind v4 + Supabase, deployed on Vercel.

## Env
Copy `.env.example` to `.env` and fill in:
- `PUBLIC_SUPABASE_URL`
- `PUBLIC_SUPABASE_ANON_KEY`
- `PUBLIC_SITE_URL`

## Dev
```bash
npm install --legacy-peer-deps
npm run dev
```
